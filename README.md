# Chenyu's Blog

基于 [Jekyll Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) 主题搭建的个人技术博客。

## 写作

文章放在 `_posts/` 目录下，文件名格式为 `YYYY-MM-DD-标题.md`。

文件头部需包含：

```yaml
---
title: 文章标题
date: YYYY-MM-DD HH:MM:SS +0800
categories: [分类名称]
tags: [标签1, 标签2]
---
```

正文使用 Markdown 语法书写。

## 部署

推送至 main 分支后，GitHub Actions 自动构建并部署到 GitHub Pages。
